# Emergency Response Predictor

Machine Learning based system that detects emergency type and suggests nearest emergency services.

## Features

- Emergency classification using ML
- Nearest ambulance / police / fire station detection
- Real time location detection
- Emergency response prediction

## Datasets

1. Emergency Room Dataset
2. 911 Emergency Calls Dataset
3. Ambulance Response Dataset
4. Service Location Dataset

Datasets are not included due to large size.

Download them from Kaggle and place inside /data folder.

## Installation

pip install -r requirements.txt

Train model

python ml_model/train_model.py

Run backend

python backend/app.py

Open frontend/index.html
